==========
Chessboard
==========

A simple chessboard implementation in Qt to learn QML and play online chess.

Features
========

None.

Build Instructions
==================

Chessboard uses `CMake`_::

    mkdir build
    cd build
    cmake -GNinja ..
    ninja

.. _CMake: http://www.cmake.org/
